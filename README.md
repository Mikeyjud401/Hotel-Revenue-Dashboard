# Hotel-Revenue-Dashboard

## Project Overview
This project analyzes sales data from multiple hotel datasets using Microsoft Excel and importing them into SQL. 
Then using Power BI to create an interactive dashboard to track revenue performance across countries.
The goal of the dashboard is to help management quickly understand sales performance, 
profitability, and trends across countries and products using SQL and Power BI.

This project was created as part of a data analysis portfolio to demonstrate SQL query writing, 
data summarization, and Power BI dashboarding for business insights. 

---

## Business Questions Answered
- Is our revenue growing by year?
- Should we increase our parking lot size?
- What trends can we see in the data?

---

## Key KPIs
- Total Revenue
- Hotel Type
- Parking Percentage
- Average Daily Rate
- Total Nights Stayed

---

### Tools used
- Microsoft Excel
- SQL Server Management Studio 22
- Power BI

## Key Insights
- Overall revenue is growing by year. 53% of the total revenue is city hotels.
- With the parking percentage reflecting a steady average of 1% for city hotels and 3% for resort hotels.
  The parking lot size does not have enough evidence to support that it should be increased in size.
- We can also see positive trends in the data such as the Average daily rate, Total nights stayed, and the Discounts.
